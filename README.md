# Lessons-in-Rxandaphael-language-

(i) Lessons in Rxandaphael language:

**"Yize ne Etkal Zenkariel ne Zara Ashva"**

**Word by word:**
- **Yize** = light
- **ne** = of
- **Etkal** = God
- **Zenkariel** = guides
- **ne Zara** = my
- **Ashva** = soul

**Translation: "The light of God guides my soul"** )


(ii) Lessons in Rxandaphael language:
**"Elivc Ashnir ne Etkal Nirthon no ne Zara Ashva, nirvariumavar Zara en Yize en Blen, en Lumara Se-zula Varlum ne Etkal Varielum-ah"**

**Word by word breakdown:**

| Rxandaphael | English |
|---|---|
| Elivc | Holy |
| Ashnir | Spirit |
| ne Etkal | of God |
| Nirthon | descends |
| no | upon |
| ne Zara Ashva | my soul |
| nirvariumavar | fills |
| Zara | me |
| en Yize | with light |
| en Blen | and power |
| en | and |
| Lumara | we |
| Se-zula | praise |
| Varlum | the kingdom |
| ne Etkal | of God |
| Varielum-ah | forever |

**Translation:**
**"The Holy Spirit of God descends upon my soul, fills me with light and power, and we praise the kingdom of God forever"** )
